# Teensy firmware

Teensy 4.1 firmware for timing telemetry and measurement.

## Current role
- Analytics / telemetry / measurement
- Sends telemetry to Raspberry Pi
- Not yet disciplining the Raspberry Pi timing chain

## Build environment
- Arduino IDE with Teensy support
- Target board: Teensy 4.1

## Main sketch
- firmware/teensy_telemetry/teensy_telemetry.ino

## Planned future roles
- FE-5680A discipline support
- Improved PPS measurement
- ZED-F9T migration support
